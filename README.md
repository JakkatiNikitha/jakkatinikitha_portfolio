# JakkatiNikitha Portfolio

This is a Vite + React + TypeScript portfolio template that uses Tailwind CSS and Supabase (optional).

Quick start

1. Install dependencies

```powershell
npm install
```

2. Create a local environment file

Copy `.env.example` to `.env` and set your Supabase variables (if you use Supabase):

```powershell
copy .env.example .env
# then edit .env and replace the placeholders
```

3. Start the dev server

```powershell
npm run dev
```

Open http://localhost:5173/ in your browser.

Notes and troubleshooting

- Tailwind CSS is configured with `tailwind.config.cjs` and `postcss.config.cjs`. If your editor shows `Unknown at rule @tailwind`, install the VS Code extension "Tailwind CSS IntelliSense" and/or add the recommended editor settings. A `.vscode/settings.json` is included to silence common `@tailwind` warnings.

- If you use Supabase features, set `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY` in `.env`.

- To build for production:

```powershell
npm run build
npm run preview
```
